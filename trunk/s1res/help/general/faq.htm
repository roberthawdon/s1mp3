Frequently Asked Questions (FAQ)

How to change my player's images?

Run s1res and choose the "Import Firmware..." from the "File" menu and follow the wizard. Once completed, you should be able to see your player's firmware content. Now edit the files before you run the "Export Firmware..." wizard, which brings the changes back to your player.

I opened an image for editing, but whenever i choose "Close" from the "Edit" menu, nothing happens!?

The close function isn't able to find and close the external image editor, which was used to open your image resource. Anyway, it closes the file internally by finally reading any file changes. If the resource is still in use, further changes to the file won't get noticed.

I see the resource icons inside the right window of s1res, but how do i edit them?

The usual way should be to double-click them. Instead you could also highlight one or multiple icons and choose "Open" from the "Edit" menu, or press RETURN alternatively. To open images, an external tool is needed to edit the provided bitmap data. First make sure some image editor capable of bitmaps (BMP) is installed. Then use the "Options" dialog to choose your preferred application. Another way may be to drag'n'drop bitmap images directly into s1res. This action depends on the source filename and replaces the first equal named resource automatically. A quick way to apply new themes to your player since you could drag'n'drop a whole bunch of files at once.

How to edit string resources?

Just select the string resource in the left window and the right window will display one or more strings. Multiple strings represent the same expression for different languages. To edit one entry, just select the appropriate line and press the left mouse button or RETURN. Inside edit mode, the status bar displays the number of available characters. The total amount is limited for each string. If you need more space then try to delete characters from an unused language entry (just select the string and press DEL).

With v3.4, s1res supports several multi-byte character code-pages. E.g. to edit some russian text, first select the proper code page. In this case go to the "Select Code Page" item inside the "Edit" menu and choose "Cyrillic (1251)". Now you should be able to edit string resources using all characters available inside this code page. If you want to learn more about windows code pages then please visit this page.

To edit non-displayable characters you can still use URL-encoded (RFC 1738) edit-mode. Go to the "Select Code Page" item inside the "Edit" menu and choose "Default (URL-encoded)". During edit, non-printable ASCII characters get encoded by a '%'-prefix, followed by a two-digit hexadecimal value, representing the ASCII-code.

I don't know how to leave s1res?

Choose "Exit" from the "File" menu or quickly press ALT+F4.
